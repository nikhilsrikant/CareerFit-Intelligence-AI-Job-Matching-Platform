# Domain-Agnostic Matching

CareerFit does not require a fixed career domain. It builds a profile from the user's provided sources and extracts skills, keywords, domains, and role phrases dynamically.

It can support software, data, electrical, mechanical, operations, design, finance, marketing, healthcare, and other domains as long as the uploaded document or website contains readable text.

For best results, upload a text-readable PDF or a real Word `.docx` file. Apple Pages documents renamed to `.docx` are not valid Word files and should be exported as Word or PDF first.
